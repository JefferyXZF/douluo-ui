# 按需引入

借助 babel-plugin-componet 我们可以只引入需要的组件，以达到减小项目体积的目的。

首先，安装 babel-plugin-component：

npm install babel-plugin-component -D
1

在 .babel.config.js 中新增

plugins: [
  [
    'component',
    {
      libraryName: 'test-component',
      styleLibrary: {
        name: 'theme',
        base: false
      }
    }
  ]
];
